\# Lucky - Personal AI Assistant



A personal AI assistant built with Python, Flask, and Ollama.



\## Features

\- Natural language chat powered by Llama 3.2 (runs fully offline)

\- Image generation using Stable Diffusion (Phase 2)

\- Clean web UI built with Flask and HTML/CSS/JS

\- Conversation memory within sessions



\## Tech Stack

\- Python 3.14

\- Flask

\- Ollama (Llama 3.2)

\- Stable Diffusion 1.5

\- HTML/CSS/JavaScript



\## Setup

1\. Install Ollama from ollama.com and run `ollama pull llama3.2`

2\. Clone this repo

3\. Create a virtual environment: `py -m venv venv`

4\. Activate it: `venv\\Scripts\\activate`

5\. Install dependencies: `pip install -r requirements.txt`

6\. Run: `py app.py`

7\. Open http://127.0.0.1:5000
